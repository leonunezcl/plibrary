Almacena todo tu código Visual Basic en esta aplicación. 
Tiene 14 categorias para almacenarlas. Se puede importar código desde archivos 
.bas, .cls, .ctl, .frm , .pag y proyectos (.vbp). 
Tambien viene con un browser integrado para buscar código en internet. 
Se puede exportar el código a archivos de texto, texto enriquecido o hypertexto. 
Escencial para todo desarrollador Visual Basic. 

Podemos destacar :

- Importación de archivos o proyectos.
- Exporta el codigo a texto, rtf o html.
- Fácil de usar y amigable.
- Respaldo del código en formato .zip.
- Rápido de usar y aprender.
- Se puede buscar código en la libreria.
- Exportar itemes a archivos .bas o .cls
- Selección de itemes de diferentes categorias en un bookmark.
